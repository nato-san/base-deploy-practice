Before You Start

Prepare these tools

  • GitHub account
  • Remix IDE
  • Rabby or MetaMask wallet
  • ETH on Base Mainnet
  • Optional IPFS upload service such as Pinata or Lighthouse

No repository copy required

You do not need to fork, clone, or download this repository. Copy the contract code, paste it into Remix, and deploy with your own wallet.

Where is the contract code?

Open BaseDeployPractice.sol on GitHub, click the copy button, then paste the code into Remix.

Base ETH is required

Deploying on Base Mainnet uses real gas. ETH on Ethereum Mainnet is not usable for Base gas. You need ETH on Base Mainnet.

What is IPFS?

IPFS stores NFT images and metadata. Upload the metadata folder, get a CID, then replace YOUR_METADATA_FOLDER_CID before deploying.

Remix labels may look different

For Base Mainnet, use WalletConnect if Browser Extension does not work. Remix VM is only a test network and does not deploy to Base Mainnet.

Remix Deploy Guide

1. Open Remix and create the contract file
  1. Open Remix.
  2. Open BaseDeployPractice.sol on GitHub.
  3. Copy all of the contract code.
  4. In Remix, open the left File Explorer, create a new file named contracts/BaseDeployPractice.sol, and paste the code.
2. Select the file and compile the contract
  1. Click BaseDeployPractice.sol in the File Explorer so it is open in the editor.
  2. Open the Solidity Compiler tab.
  3. Select compiler version 0.8.20 or a newer 0.8.x version.
  4. Click Compile BaseDeployPractice.sol.
3. Connect your wallet to Remix
  1. Open the Deploy & Run Transactions tab.
  2. Choose WalletConnect. If your browser wallet works with Browser Extension, that is also OK.
  3. Approve the connection in Rabby or MetaMask.
4. Switch to Base Mainnet
  1. Open your wallet network selector.
  2. Choose Base Mainnet.
  3. Make sure you have a small amount of ETH on Base for gas.
  4. If Remix shows Remix VM, you are only on a test network. Switch to WalletConnect for the real Base Mainnet deployment.
5. Deploy the contract
  1. Select BaseDeployPractice in Remix.
  2. Find the initialTokenURI input box under the Deploy section.
  3. Paste "ipfs://YOUR_METADATA_FOLDER_CID/metadata.json" into that initialTokenURI box.
  4. Do not leave initialTokenURI empty. If Remix says gas estimation failed, check this field first.
  5. Click Deploy and confirm the transaction in your wallet.
6. Find and click the mint button
  1. After deployment, open Deployed Contracts in Remix.
  2. Expand BASEDEPLOYPRACTICE.
  3. Click the orange mint button and confirm the transaction.

Common Errors

Wrong network

Switch your wallet to Base Mainnet before deploying or minting.

Not enough ETH

Minting is free, but every transaction still needs gas on Base.

Already minted

Each wallet can mint only once from this beginner practice contract.
