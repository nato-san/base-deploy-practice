Lessons

1

Deploy with Remix

Compile the Solidity file, connect Rabby or MetaMask with Injected Provider, switch to Base Mainnet, and deploy the contract.

2

Verify on BaseScan

Find your deployed contract, match the compiler version, enter the constructor value, and publish the source code.

3

Mint one NFT

Use Remix's deployed contract panel, click mint(), confirm the transaction, and mint the Hamuko image as your practice NFT.

Required Tools

  • ✓ GitHub account
  • ✓ Remix IDE
  • ✓ Rabby or MetaMask wallet
  • ✓ Small amount of ETH on Base Mainnet

Common Errors

Wrong network

Switch your wallet to Base Mainnet before deploying or minting.

Not enough ETH

Minting is free, but every transaction still needs gas on Base.

Already minted

Each wallet can mint only once from this beginner practice contract.

Important: This page is only a visual learning guide. Deployment and minting are still done in Remix, exactly as described in the README.
